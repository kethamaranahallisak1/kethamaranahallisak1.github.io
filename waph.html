Web Application Programming and Hacking (WAPH) – Summer 2025

This cross-listed course CPS499-Z1/499-P1/592-Z1 is asynchronous and focuses on front-end and back-end web application development, ethical hacking, and web application security. Students engage with lecture videos, labs, and hands-on activities to build secure web systems.

Course Description and Objectives

In this course, students will study client-side development using HTML5, CSS, JavaScript, and server-side development using PHP and MySQL. Key security vulnerabilities (XSS, CSRF, etc.) and corresponding secure coding practices will be covered through hands-on labs.

Course Goals:
  • Build secure web applications using modern web technologies.
  • Improve programming skills and software robustness.
  • Analyze security flaws and implement countermeasures.
  • Develop real-world solutions and avoid known web vulnerabilities.
Learning Outcomes:
  • Design and implement secure web-based solutions.
  • Understand major web application vulnerabilities and attacks.
  • Demonstrate hands-on experience in ethical hacking simulations.
  • Adopt defensive programming to secure applications.
